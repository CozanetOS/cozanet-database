# cozanet-database

Relational & semantic vector storage layer for the Cozanet OS agent framework, powered by Supabase and PostgreSQL.

## Features

- **TypeScript Native Agents**: Modular database client agents for users, workflows, tasks, goals, and knowledge.
- **Vector Search Support**: Built-in support for embedding-based memory searches utilizing pgvector.
- **SQL Migrations**: Included full initial relational schema migrations.
